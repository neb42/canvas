# Widget styles

## Common styles

* Background color: A palette selector and color picker to choose a background color.
The palette can be set in `client.json`.
* Overflow: Defines how the widget responds to overflowing content.
`visible`, `hidden`, `auto`, or `scroll`.
* Class names: Applies a list of class names to the container tag.
* CSS: Applies arbitary CSS to the container tag.
